# Cara pemasangan

1. Gunakan NodeJs dengan versi 18 keatas
2. Jalankan perintah berikut
```bash
npm install
cp .env.example .env
```
3. Sesuaikan isi dari .env
4. Jalankan perintah berikut
```bash
npm run dev
```
